# _Invader Records_

##### _A record label for the world's greatest conquerors, 8/28/15_

#### By _**Shannon Mulloy**_

## Description

Check out when conquering bands invaded a venue near you. Take up arms and create a new band, and add venues to conquer. Edit or delete your band or venue at any time.

## Setup

* _Clone from GitHub_
* _Run: bundle_
* _Run: rake db:create_
* _Run: rake db:migrate_
* _Launch in Sinatra_

## Technologies Used

_Ruby, ActiveRecord_
